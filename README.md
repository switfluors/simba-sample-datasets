# SIMBA Sample Microscopy Datasets

This repository provides sample microscopy datasets for testing and evaluation of **SIMBA**.

## Associated preprint

**SIMBA: an agentic AI platform for single-molecule multi-dimensional imaging**  
Hongjing Mao, Harsh Mauny, Obblivignes KanchanadeviVenkataraman, Caroline Laplante, Dongkuan Xu, Yang Zhang

**DOI:** https://doi.org/10.64898/2026.04.16.719005

## About this repository

This repository is intended for **data distribution only**. It contains sample microscopy datasets associated with the SIMBA preprint.

The software source code is **not yet public** in this repository.

## Project website

Learn more about SIMBA here:

https://simba-website-indol.vercel.app/

## Dataset packages

The sample data are provided as two downloadable ZIP files in the GitHub Release assets:

### 1. SIMBA_Nuclearpore_sample_data.zip
Contents include:
- one raw `.nd2` microscopy file (~1 GB)
- related smaller files needed for demonstration and testing

### 2. SIMBA_sSMLM_sample_data.zip
Contents include:
- one raw `.nd2` microscopy file (~1 GB)
- related smaller files needed for demonstration and testing

## Download

Download the files from the latest GitHub Release:

- [SIMBA_Nuclearpore_sample_data.zip](https://github.com/YOUR-USERNAME/YOUR-REPO/releases/latest/download/SIMBA_Nuclearpore_sample_data.zip)
- [SIMBA_sSMLM_sample_data.zip](https://github.com/YOUR-USERNAME/YOUR-REPO/releases/latest/download/SIMBA_sSMLM_sample_data.zip)

After downloading, unzip them into a local data directory, for example:

```text
data/
  Nuclearpore/
    ...
  sSMLM/
    ...
```
